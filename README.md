📊 Exploratory Data Analysis (EDA) Project
🚀 Overview

This project performs in-depth Exploratory Data Analysis (EDA) on a structured sales dataset to extract actionable insights, identify patterns, and support data-driven business decisions.

The analysis focuses on sales performance, customer behavior, and product efficiency across multiple dimensions.

📂 Dataset Architecture

The dataset follows a Star Schema Model ⭐

🔹 Fact Table
fact_sales → Transactional sales data
🔹 Dimension Tables
dim_customers → Customer details 👤
dim_products → Product details 🛍️
