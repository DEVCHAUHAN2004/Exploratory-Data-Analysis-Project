# 📊 Exploratory Data Analysis (EDA) Project

## 🚀 Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on sales data to understand business performance, identify patterns, and generate useful insights.

---

## 📂 Dataset

The dataset contains information about:

* Customers 👤
* Products 🛍️
* Orders 📦
* Sales 💰

Tables used:

* `fact_sales`
* `dim_customers`
* `dim_products`

---

## 🎯 Objectives

* Analyze total sales and revenue
* Understand customer distribution
* Identify top and worst-performing products
* Explore sales trends across countries and categories

---

## 📌 Key Questions Solved

* What is the total sales?
* How many items are sold?
* What is the average selling price?
* Total number of orders, products, and customers
* Revenue generated by each category
* Revenue generated by each customer
* Distribution of sales across countries
* Top 5 products by revenue
* Bottom 5 products by performance

---

## 🛠️ Tools & Technologies

* SQL (Hive / SQL Server)
* Data Analysis Concepts
* Git & GitHub

---

## 📊 Key Insights

* Identified high revenue-generating products
* Found low-performing products
* Analyzed customer behavior across regions
* Observed category-wise performance trends
