📊 Exploratory Data Analysis (EDA) Project
<p align="center"> <b>🚀 Data-Driven Insights | SQL Analytics | Business Intelligence</b> </p>
🚀 Overview
  
This project performs Exploratory Data Analysis (EDA) on a structured sales dataset to uncover insights, identify trends, and su pport data-driven decision-making.

📂 Dataset Architecture 

⭐ Star Schema Model

🟦 fact_sales → Transaction data
🟩 dim_customers → Customer details 👤
🟨 dim_products → Product details 🛍️
🎯 Objectives

✔ Analyze sales performance
✔ Understand customer behavior
✔ Identify top & low-performing products
✔ Study country-wise trends

📊 Key Insights

🚀 High revenue products identified
⚠️ Underperforming products detected
🌍 Regional trends analyzed
👥 Customer contribution evaluated

📌 Business Questions
💰 Total revenue generated
📦 Total items sold
💵 Average selling price
🛍️ Category-wise revenue
🌍 Country-wise sales
🏆 Top & bottom products
